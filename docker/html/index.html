🏗️ Microservices Architecture

Sistema de gestión de inventario y órdenes con comunicación asíncrona

📦 Servicio de Inventario

Gestiona el stock de productos, movimientos de inventario y alertas de stock bajo.

  • 🔍 Health Check
  • 📋 Productos
  • ⚠️ Stock Bajo

🛒 Servicio de Órdenes

Procesa órdenes de compra y venta, gestiona estados y se comunica con inventario.

  • 🔍 Health Check
  • 📋 Órdenes
  • 📊 Estadísticas

📊 Servicio de Monitoreo

Monitorea la salud de todos los servicios y proporciona métricas del sistema.

  • 🔍 Health Check
  • 🏥 Estado de Servicios
  • 📈 Dashboard
  • 🚨 Alertas

🛠️ Herramientas de Desarrollo

📊 Kafka UI 🐘 PostgreSQL 📈 Estadísticas Kafka 💚 Estado del Sistema

🚀 Endpoints de API

GET /api/inventory/products GET /api/orders/orders GET /api/monitor/dashboard

Arquitectura de Microservicios con Flask, Kafka, PostgreSQL y Docker

🐍 Python | 🐳 Docker | 📨 Kafka | 🐘 PostgreSQL
